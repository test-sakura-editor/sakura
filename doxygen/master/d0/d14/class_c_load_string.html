◆ LoadStringSt()

LPCWSTR CLoadString::LoadStringSt ( UINT  uid)
static

静的バッファに文字列リソースを読み込む（各国語メッセージリソース対応）

引数
[in]uidリソースID
戻り値
読み込んだ文字列（文字列無しの時"" が返る）
覚え書き
静的バッファ（m_acLoadStrBufferTemp[?]）に文字列リソースを読み込む。
バッファは複数準備しているが、呼び出す毎に更新するのでバッファ個数を
超えて呼び出すと順次内容が失われていく。
呼び出し直後での使用や関数の引数などでの使用を想定しており、前回値を
取り出すことはできない。
使用例）::SetWindowText( m_hWnd, CLoadString::LoadStringSt(STR_ERR_DLGSMCLR1) );
アプリケーション内の関数への引数とする場合、その関数が本関数を使用
しているか意識する必要がある（上限を超えれば内容が更新されるため）
内容を保持したい場合は CLoadString::LoadString() を使用する。
