int CLoadString::CLoadStrBuffer::LoadString ( UINT  uid)

文字列リソースを読み込む（読み込み実行部）

引数
[in]uidリソースID
戻り値
読み込んだ文字数（TCHAR単位）
覚え書き
メッセージリソースより文字列を読み込む。メッセージリソースDLLに指定の
リソースが存在しない、またはメッセージリソースDLL自体が読み込まれて
いない場合、内部リソースより文字列を読み込む。
最初は静的バッファに読み込むがバッファ不足となったらバッファを拡張
して読み直す。
取得したバッファはデストラクタで解放する。
ANSI版は2バイト文字の都合により（バッファ - 2）バイトまでしか読まない
場合があるので1バイト少ない値でバッファ拡張を判定する。
