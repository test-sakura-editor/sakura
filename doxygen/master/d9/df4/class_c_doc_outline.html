Python アウトライン解析 // 2007.02.08 genta

python関数リスト作成

class, def で始まる行から名前を抜き出す．

class CLASS_NAME( superclass ): def FUNCTION_NAME( parameters ):

文字列とコメントを除外する必要がある．

通常の行頭の場合に関数・クラス判定と登録処理を行う． Python特有の空白の数を数えてネストレベルを判定する． indent_levelを配列として用いており，インデントレベルごとのスペース数を格納する． なお，TABは8桁区切りへの移動と解釈することになっている．

通常の行頭でない(文字列中および継続行)，あるいは行頭の処理完了後は 状態機械 python_analyze_state に判定処理を依頼する．

文字列
'' "" 両方OK 引用符3連続でロング文字列 直前にrかRがあったらエスケープ記号を無視(ただし改行のエスケープは有効)
コメント
#で始まり，行の継続はない．
