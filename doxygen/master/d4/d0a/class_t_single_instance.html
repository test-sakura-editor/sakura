static T * getInstance ()
 作成済みのインスタンスを返す。インスタンスが存在しなければ NULL。
 

詳解

template<class T>
class TSingleInstance< T >

1個しかインスタンスが存在しないクラスからのインスタンス取得インターフェースをstaticで提供。 Singletonパターンとは異なり、Instance()呼び出しにより、インスタンスが自動生成されない点に注意。
