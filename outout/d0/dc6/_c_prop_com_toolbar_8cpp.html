Owner Draw List Boxに指定の値を追加する (Windows XPの問題回避用)

Windows XP + manifestの時にLB_ADDSTRINGが値0を受け付けないので とりあえず0以外の値を入れてから0に設定し直して回避する。 1回目の挿入は0でなければ何でもいいはず。

引数

index[in] 挿入位置
value[in] 挿入する値
戻り値
挿入位置。エラーの時はLB_ERRまたはLB_ERRSPACE
