◆ MakeSureDirectoryPathExistsW()

BOOL ApiWrap::MakeSureDirectoryPathExistsW ( LPCWSTR  szDirPath)

MakeSureDirectoryPathExists の UNICODE 版。 szDirPath で指定されたすべてのディレクトリを作成します。 ディレクトリの記述は、ルートから開始します。

引数
DirPath有効なパス名を指定する、null で終わる文字列へのポインタを指定します。 パスの最後のコンポーネントがファイル名ではなくディレクトリである場合、 文字列の最後に円記号（\）を記述しなければなりません。
戻り値
関数が成功すると、TRUE が返ります。 関数が失敗すると、FALSE が返ります。
覚え書き
指定された各ディレクトリがまだ存在しない場合、それらのディレクトリを順に作成します。 一部のディレクトリのみを作成した場合、この関数は FALSE を返します。
